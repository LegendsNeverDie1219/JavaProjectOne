# 注解
## 一.背景
1.JDK5.0开始,增加了对注解(元数据)的支持
2.Annotation就是代码里面的特殊标记, 这些标记可以类编译,class加载,以及执行过程被读取.并且执行相应的逻辑处理
目的可以代替之前的JAVAEE中繁杂代码和xml配置.
3.Annotation可以像修饰符一样的修饰, 修饰 类, 构造器, 属性,方法,局部变量
4.注解是未来开发模式的一种趋势,像现在流行的Spring2.5以上都是基于注解.
框架: 注解 +反射 +设计模式.
